# Base de connaissances TDAH

Cette base de connaissances regroupe des informations éducatives sur le TDAH (Trouble du Déficit de l'Attention avec ou sans Hyperactivité) et les sujets connexes. Ces données servent de référence pour enrichir l'application Tokō et aider les parents à mieux comprendre et accompagner leurs enfants.
